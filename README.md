# TaskNest 🪺

**TaskNest** is a personal productivity dashboard built with the MERN Stack (MongoDB, Express, React, Node.js).  
This project includes full authentication, task management, goals tracking, and personal notes.

## Features
- User Authentication (JWT)
- Task Manager (CRUD)
- Goal Tracker
- Notes Section
- Dashboard with summary
- Fully responsive UI

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- Auth: JWT
